Optical Character Recognition (OCR) converts images of text — such as scanned pages — into machine-readable text.

This app automatically detects pages that contain scanned images instead of selectable text and applies OCR via Tesseract to extract the content.

Tip: OCR works best on clean, high-resolution scans. Handwritten or heavily stylized text may produce lower accuracy.

  • File type: PDF only (.pdf)
  • Text PDFs: Searchable instantly using native text extraction
  • Scanned PDFs: Automatically processed with OCR (may take longer)
  • Storage: Only one PDF is kept at a time — uploading a new file replaces the previous one
  • Privacy: All processing happens locally on your machine. No data is sent externally.

Q: Why is the search slow?
A: Pages without selectable text are processed with OCR, which can take several seconds per page. Native-text PDFs search almost instantly.

Q: Can I upload multiple files?
A: Currently, only one PDF is stored at a time. Uploading a new file automatically replaces the previous one.

Q: Is my data secure?
A: Yes — everything runs locally on your machine. No files or data are uploaded to any external server.

Flask PyMuPDF Tesseract OCR Bootstrap 5

Built with Python + Flask on the backend. PDFs are parsed using PyMuPDF (fitz). Scanned pages are handled via Tesseract OCR. The frontend uses Bootstrap 5 with custom styling.
